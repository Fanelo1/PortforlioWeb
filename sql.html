Project 2: Applying Filters to SQL Queries for Security

This project focuses on using SQL with filters to perform security-related tasks and investigate potential security issues.

Key Tasks:

  • Retrieve after-hours failed login attempts
  • Retrieve login attempts on specific dates
  • Retrieve login attempts outside of Mexico
  • Retrieve employees in specific departments (Marketing, Finance, Sales)
  • Retrieve all employees not in IT

This project demonstrates the application of SQL filters to extract specific information from log_in_attempts and employees tables, using various operators like AND, OR, NOT, and LIKE for precise data retrieval.
