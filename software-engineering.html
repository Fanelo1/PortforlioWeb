University Study Management System

A comprehensive solution for students to manage their academic life

Project Overview

The University Study Management System is designed to help students organize their study schedules, track assignments, and prepare for exams. This centralized platform aims to reduce stress, increase productivity, and boost academic performance.

Key Features:

  • Mobile application for student portal
  • Web application for admin panel
  • Study schedule management
  • Note management and organization
  • Revision tools and study aids
  • Internal messaging system

Technical Specifications

Development Tools:

  • Mobile App: Flutter
  • Web App: ASP.NET Core (Razor Pages)
  • Backend: ASP.NET Core
  • Database: Firebase Firestore
  • Authentication: Firebase Authentication
  • Notifications: Firebase Cloud Messaging

The system is designed to handle up to 3000 Daily Active Users and maintain 99.99% uptime.

Project Deliverables

  • Mobile application for the student portal
  • Web application for the admin panel
  • Centralized backend system with Firebase integration
  • User documentation and system manual

Project Timeline

The project is scheduled to be completed within 12 weeks, with a team of 6 members and a budget of R928.65.

Key Milestones:

  • Requirements gathering and analysis
  • System design and prototyping
  • Development of mobile and web applications
  • Integration with Firebase backend
  • Testing and quality assurance
  • Deployment and user training
